COMMON HTML TAG

    1.Tags
    • Formatting Tags:
      •
      A formatting tag in HTML is a tag that changes the appearance of text, like making it bold, italic, underlined, or highlighted.

    • Semantic Tags:
      •
      Semantic tags are HTML tags that describe the meaning or purpose of the content they enclose.

    • List Tags:
      •
      A list tag in HTML is used to organize items in a list, either ordered (`ol`), unordered (`ul`), or as a description list (`dl`)

    • Link & Media Tags:
      •
      Link and media tags in HTML are used to add hyperlinks, images, audio, video, or embedded content to a webpage

    • Table Tag:
      • A table tag in HTML is used to create a table to organize data in rows and columns
